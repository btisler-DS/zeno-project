# Zeno Non-Commercial License — Compliance Guide
Version 0.2 • Updated: 2025-11-23

This guide provides a plain-language summary of what users *can* and *cannot* do with Zeno under the ZNL terms.

---

# ✔ Allowed Uses (Non-Commercial)

- Personal research  
- Academic or educational work  
- Non-profit research  
- Open-science model evaluation  
- Student projects  
- Benchmarking open models for scientific publication  

All allowed uses must be **non-commercial** and **non-competitive**.

---

# ❌ Prohibited Uses (Commercial)

The following require a paid licensing agreement:

- Internal or external model evaluation by a for-profit organization  
- Using Zeno to benchmark commercial APIs or internal models  
- Integrating Zeno into CI/CD, QA, or any corporate workflow  
- Using Zeno outputs in:
  - investor decks  
  - marketing  
  - customer materials  
  - sales pitches  
- Using Zeno to support paid consulting or client services  
- Publishing Zeno-derived model scores in commercial blogs, reviews, or whitepapers

If the output benefits a company, the use is commercial.

---

# ✔ Permitted With Commercial License

- Enterprise R&D  
- Consulting or analysis services  
- Embedding Zeno into automation pipelines  
- Creating dashboards or SaaS interfaces  
- Publishing model rankings with Zeno scores  
- Using Zeno as part of model-development lifecycles  

---

#  Enforcement Reminder

Violating the license triggers **automatic termination**, after which *any* further use becomes copyright infringement.

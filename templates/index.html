⚡ Zeno Calibration Harness

Black-box calibration for large language models

Zeno does not run models. You bring the model endpoint and API key. All computation stays on your provider’s infrastructure.

Run New Calibration

Quick start: Use an OpenAI-style endpoint such as https://api.openai.com/v1/chat/completions with a model like gpt-4.1, gpt-4o, or o3-mini.

For local LM Studio / MstyStudio, this only works when running Zeno on the same machine. In the hosted demo, use a public or cloud API endpoint.
Any provider-specific model ID (e.g. gpt-4.1, gpt-4o, deepseek-r1).
Keys are sent only to your model provider. Zeno does not log or store them.

Advanced: Providers like DeepSeek or OpenAI-compatible gateways in front of Claude / Gemini / HuggingFace can also be used here. Zeno is provider-agnostic as long as the API speaks the OpenAI chat format.

Live calibration is disabled in this hosted demo. Clone the repository or set ZENO_DEMO_MODE=false to run full calibrations on your own infrastructure.
